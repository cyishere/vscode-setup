## Settings

- Autoformatting
  - Format on save

- Font
  - Font Ligatures: true

- Enable Emmet support for JSX
  - Settings -> User -> Extensions -> Emmet, add following snippet to the json file:
  
  ```json
  {
    "emmet.includeLanguages": {
      "javascript": "javascriptreact"
     }
  }
  ```
